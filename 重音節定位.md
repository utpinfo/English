# 重音固定在「倒數第 2 音節」
| 詞尾                | 例子                           |
| ----------------- | ---------------------------- |
| **-tion / -sion** | edu**CA**tion, deci**SION**  |
| **-ic**           | geo**GRA**phic, dra**MA**tic |
| **-ial / -ual**   | fi**NAN**cial, ha**BI**tual  |
| **-ity**          | a**BIL**ity, elec**TRI**city |
| **-graphy**       | pho**TO**graphy              |


# 重音固定在「最後一音節」
| 詞尾       | 例子                        |
| -------- | ------------------------- |
| **-ee**  | emplo**YEE**, refu**GEE** |
| **-eer** | engi**NEER**              |
| **-ese** | Japa**NESE**              |


# 名詞 vs 動詞：同字不同命
| 詞性           | 重音         |
| ------------ | ---------- |
| **名詞 / 形容詞** | **第 1 音節** |
| **動詞**         | **第 2 音節** |


# 前綴通常「不吃重音」
a-, be-, con-, de-, dis-, ex-, re-, pre-, pro-

# 長音節 > 短音節（更容易吃重音）
- 長母音 / 雙母音 (ai, ee, oa…)
- 結尾是子音群 (–ct, –nd, –st)
- 音節比較「重」


# 結論
先看詞尾 → 再看詞性 → 忽略前綴 → 重音給「比較重」的音節

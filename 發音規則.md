發音規則 90%

# 結論
- 開音長 (母音結尾)
- 閉音短 (子音結尾)

# 開閉音判斷法
- 0子音 = 發長音 -> hi
- 1子音 = 發短音 -> at
- 2子音 = 無子音 -> hello
- 子音 + e結尾 = 1子音 -> ate

# 字母弱化
- 非重音節
- 字首 a/e/o (95%弱化)
- 字尾 -er -or -al -on -tion -ment (100%弱化)


## 長音 (唸字母名)
- 母音後面「沒子音」
- 母音後面「只有一個子音＋e」
hi, go, me, no → /oʊ, iː, aɪ, oʊ/
hire, cake, bike, home → /keɪk, baɪk, hoʊm/

## 短音 (次音)
- 母音後面[有兩個以上子音]
- 母音後面[字尾沒e]
cat, bed, big, hot, cup → /kæt, bed, bɪɡ, hɑt, kʌp/
apple, little, bottle → 短音



## 多音節(短音弱化成)
- 字首在重讀音節 → 保持原短 e 發音 /ɛ/
- 字首在非重讀音節 → 弱讀成 /ə/

| 單字         | bene- 發音 | 原因           |
| ---------- | -------- | ------------ |
| benefactor | /bɛn/    | 重讀音節，閉音節短 e  |
| benevolent | /bə/     | 非重讀音節，弱讀 /ə/ |
